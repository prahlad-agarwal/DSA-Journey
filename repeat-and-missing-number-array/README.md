<h2><a href="https://www.interviewbit.com/problems/repeat-and-missing-number-array">Repeat and Missing Number Array</a></h2> <img src='https://img.shields.io/badge/Difficulty-Medium-orange' alt='Difficulty: Medium' /><hr><p>You are given a read only array of n integers from 1 to n.</p>

<p>Each integer appears exactly once except A which appears twice and B which is missing.</p>

<p>You must write an algorithm with <code>O(log n)</code> runtime complexity.</p>

<p>Return A and B.</p>

<p>&nbsp;</p>

<i>Note: Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?</i>

<p>&nbsp;</p>

<i>Note that in your output A should precede B.</i>

<p>&nbsp;</p>
<p><strong class="example">Example:</strong></p>
<pre><strong>Input:</strong>[3 1 2 5 3]
<strong>Output:</strong>[3, 4]
</pre><p>&nbsp;</p>
<p>A = 3, B = 4</p>
