<h2><a href="https://www.geeksforgeeks.org/problems/count-total-set-bits-1587115620/1">Count set bits in 1 to n</a></h2> <img src='https://img.shields.io/badge/Difficulty-Medium-orange' alt='Difficulty: Medium' /><hr>

<p>Given a number <code>n</code>. Return the total count of set bits for all numbers from 1 to n (both inclusive).</p>

<p><strong class="example">Examples:</strong></p>

<pre>
<strong>Input:</strong> n = 4
<strong>Output:</strong> 5
<strong>Explanation:</strong> For numbers from 1 to 4. for 1: 0 0 1 => 1 set bit, for 2: 0 1 0 => 1 set bit, for 3: 0 1 1 => 2 set bits, for 4: 1 0 0 => 1 set bit. Therefore, the total set bits are 5.
</pre>

<p>&nbsp;</p>

<pre>
<strong>Input:</strong> n = 17
<strong>Output:</strong> 35
<strong>Explanation:</strong> From numbers 1 to 17(both inclusive), the total number of set bits are 35.
</pre>

<p>&nbsp;</p>

<p><strong>Constraints:</strong></p>

<ul>
	<li><code>1 &lt;= n &lt;= 10<sup>8</sup></code></li>
</ul>
