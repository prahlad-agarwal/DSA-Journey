<h2><a href="https://www.geeksforgeeks.org/problems/set-bits0143/1">Count Set Bits</a></h2> <img src='https://img.shields.io/badge/Difficulty-Easy-brightgreen' alt='Difficulty: Easy' /><hr>

<p>Given a positive integer <code>n</code>. Your task is to return the <code>count</code> of set bits.</p>

<p><strong class="example">Examples:</strong></p>

<pre>
<strong>Input:</strong> n = 6
<strong>Output:</strong> 2
<strong>Explanation:</strong> Binary representation is '110', so the count of the set bit is 2.
</pre>

<p>&nbsp;</p>

<pre>
<strong>Input:</strong> n = 8
<strong>Output:</strong> 1
<strong>Explanation:</strong> Binary representation is '1000', so the count of the set bit is 1.
</pre>

<p>&nbsp;</p>

<p><strong>Constraints:</strong></p>

<ul>
	<li><code>1 &lt;= n &lt;= 10<sup>9</sup></code></li>
</ul>
