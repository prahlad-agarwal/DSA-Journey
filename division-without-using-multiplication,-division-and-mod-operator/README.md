<h2><a href="https://www.geeksforgeeks.org/problems/division-without-using-multiplication-division-and-mod-operator/1">Division without using multiplication, division and mod operator</a></h2> <img src='https://img.shields.io/badge/Difficulty-Medium-orange' alt='Difficulty: Medium' /><hr>

<p>Given two integers <code>a</code> and <code>b</code>, find the <code>quotient</code> obtained by dividing a by b <code>without</code> using multiplication (*), division (/), or modulus (%) operators.</p>

<p>The quotient should be truncated toward zero. Return the resulting quotient.</p>

<p><strong class="example">Examples:</strong></p>

<pre>
<strong>Input:</strong> a = 10, b = 3
<strong>Output:</strong> 3
<strong>Explanation:</strong> 10/3 gives quotient as 3 and remainder as 1.
</pre>

<p>&nbsp;</p>

<pre>
<strong>Input:</strong> a = 43, b = -8
<strong>Output:</strong> -5
<strong>Explanation:</strong> 43/(-8) gives quotient as -5 and remainder as 3.
</pre>

<p>&nbsp;</p>

<p><strong>Constraints:</strong></p>

<ul>
	<li><code>-10<sup>9</sup> &lt;= a, b &lt;= 10<sup>9</sup></code></li>
</ul>
