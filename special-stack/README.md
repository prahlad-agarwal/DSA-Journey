<h2><a href="https://www.geeksforgeeks.org/problems/special-stack/1">Special Stack </a></h2> <img src='https://img.shields.io/badge/Difficulty-Easy-brightgreen' alt='Difficulty: Easy' /><hr>

<p>Design a data-structure SpecialStack that supports all the stack operations like <code>push(), pop(), isEmpty(), isFull()</code> and an additional operation <code>getMin()</code> which should return the <code>minimum</code> element from the SpecialStack. Your task is to complete all the functions, using a stack data structure.</p>

<p><code>Note:</code> The output of the code will be the value returned by <code>getMin()</code> function.</p>

<p><strong class="example">Examples:</strong></p>

<pre>
<strong>Input:</strong> stack: 18 19 29 15 16
<strong>Output:</strong> 15
<strong>Explanation:</strong> The minimum element of the stack is 15.
</pre>

<p>&nbsp;</p>

<pre>
<strong>Input:</strong> tack: 34 335 1814 86
<strong>Output:</strong> 34
<strong>Explanation:</strong> The minimum element of the stack is 34.
</pre>

<p>&nbsp;</p>

<p><strong>Constraints:</strong></p>

<ul>
	<li><code>1 &lt;= ele &lt;= 10<sup>5</sup></code></li>
	<li><code>1 &lt;= stack.size() &lt;= 10<sup>4</sup></code></li>

</ul>
